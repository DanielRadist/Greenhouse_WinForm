\doxysubsection{Подробное описание}
Строитель системы контроля климата 

Дочерний класс от интерфеса \mbox{\hyperlink{class_i_system_builder}{ISystem\+Builder}}. Содержит в себе реализацию некоторых виртуальных методов, необходимых для создания \mbox{\hyperlink{class_system_control}{System\+Control}}. Является частью паттерна \char`\"{}Строитель\char`\"{}.
