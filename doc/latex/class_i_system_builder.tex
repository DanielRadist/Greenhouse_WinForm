\doxysubsection{Подробное описание}
Интерфес строителя систем\+: \mbox{\hyperlink{class_system_sensor}{System\+Sensor}} и \mbox{\hyperlink{class_system_control}{System\+Control}}. 

Интерфес описывает виртуальные метода для дочерних классов. Подразумевается, что дочерний класс некоторые из виртуальных методов может не реализовывать. Является частью паттерна \char`\"{}Строитель\char`\"{}.
