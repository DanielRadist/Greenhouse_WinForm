\doxysubsection{Подробное описание}
Директор классов строителей унаследованных от интерфеса \mbox{\hyperlink{class_i_system_builder}{ISystem\+Builder}}. 

Данный класс использует методы дочерних классов от интерфеса \mbox{\hyperlink{class_i_system_builder}{ISystem\+Builder}} для создания систем климата теплицы. Является классом одиночкой. Для получения объекта используйте статический метод \mbox{\hyperlink{class_system_director_a98295882d87874f008047fb884a42421}{Instance()}}. Является частью паттерна \char`\"{}Строитель\char`\"{}.
