\doxysubsection{Подробное описание}
Интерфес контроллеров. 

Данный класс описывает два виртуальных метода для дочерних контроллеров, реализация которых выполнена в дочерних классах, а также выступает как абстрактный класс\+: имеет поле типа Type\+Control, которое можно получить с помошью метода get\+Type().
