Подробное описание

Интерфес строителя систем: SystemSensor и SystemControl.

Интерфес описывает виртуальные метода для дочерних классов. Подразумевается, что дочерний класс некоторые из виртуальных методов может не реализовывать. Является частью паттерна "Строитель".
