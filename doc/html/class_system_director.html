Подробное описание

Директор классов строителей унаследованных от интерфеса ISystemBuilder.

Данный класс использует методы дочерних классов от интерфеса ISystemBuilder для создания систем климата теплицы. Является классом одиночкой. Для получения объекта используйте статический метод Instance(). Является частью паттерна "Строитель".
