Подробное описание

Система контроллеров. Связующая часть между MainSystem и дочерними классами от IControl.

Объект данного класса хранит внутри приватного поля список с указателями на объекты DeviceControl. Имеет методы для работы с ранее упомянутыми объектами. Занимается управлением состояний связанных с объектом данного класса объектов DeviceControl.
