Подробное описание

Система сенсоров. Связующая часть между MainSystem и дочерними классами от Sensor.

Объект данного класса хранит внутри приватного поля список с указателями на объекты дочерних классов от Sensor. Имеет методы для работы с ранее упомянутыми объектами. Занимается получением данных с объектов дочерних классов от Sensor.
