Подробное описание

Главная система управления климатом.

Является связующим звеном между SystemSensor и SystemControl. Принимает решение по управлению микроклиматом в теплице на основе стратегий от интерфеса IStrategy. Имеет доступ к метеостанции. Хранит данные о текущей выращиваемой культуры.
