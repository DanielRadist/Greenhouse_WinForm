Подробное описание

Строитель системы датчиков (сенсоров) климата

Дочерний класс от интерфеса ISystemBuilder. Содержит в себе реализацию некоторых виртуальных методов, необходимых для создания SystemSensor. Является частью паттерна "Строитель".
